TECHNOLOGY WE USE

library & Aplications

Pygame

Our game is developed using Pygame, a versatile Python library for creating games and multimedia applications. Pygame offers efficient tools for handling graphics, sound, and animations, allowing us to manage key elements like character movement, collision detection, and user input. This technology gives us full control over game functionality, making it a perfect fit for our development needs.

Tiled

We utilize Tiled, a flexible and feature-rich map editor, to design and organize the game’s levels and environments. Tiled allows us to create detailed tile-based maps and manage various in-game elements, making level design efficient and visually cohesive. With Tiled, we can precisely customize layouts and gameplay elements to fit the game’s narrative and challenges.

Visual Studio

We use Visual Studio as our primary integrated development environment (IDE) for game development. Visual Studio provides robust tools for writing, debugging, and managing our code, allowing us to streamline development and ensure high-quality performance. Its powerful features enhance productivity and help maintain an organized and efficient workflow throughout the project.
